## Como publicar a pagina em construção:

1 - Clone

	git clone https://github.com/janasouza/marcando-de-batom.git

2 - Install all dependencies:

	$ cd marcando-de-batom/
	$ npm install

3 - Deploy for Github pages:

	$ grunt deploy
